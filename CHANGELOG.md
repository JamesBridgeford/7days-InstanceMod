## Version Format

- **Major**: Breaking changes or major feature releases
- **Minor**: New features, backward compatible
- **Build**: Bug fixes and minor improvements
- **Revision**: Hot fixes and patches

Example: `1.2.3.4` = Major.Minor.Build.Revision
